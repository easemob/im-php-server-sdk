\item {\ttfamily username} String 类型，用户名，长度不可超过 64 个字节长度。支持以下字符集：
\begin{DoxyItemize}
\item 26 个小写英文字母 a-\/z；
\item 26 个大写英文字母 A-\/\+Z；
\item 10 个数字 0-\/9；
\item “\+\_\+”, “-\/”, “.\+”。 
\begin{DoxyPre}{\bfseries{注意：不区分大小写。同一个 app 下，用户名唯一。}}\end{DoxyPre}

\end{DoxyItemize}
\item {\ttfamily password} String 类型，登录密码，长度不可超过 64 个字符长度。
\item {\ttfamily nickname} String 类型，昵称（可选），仅用在客户端推送通知栏显示的昵称，并不是用户个人信息的昵称，开发者可自定义该内容。长度不可超过 100 个字符。支持以下字符集：
\begin{DoxyItemize}
\item 26 个小写英文字母 a-\/z；
\item 26 个大写英文字母 A-\/\+Z；
\item 10 个数字 0-\/9；
\item 中文；
\item 特殊字符。 
\end{DoxyItemize}
\end{DoxyItemize}\\
